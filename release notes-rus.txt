Версия: 1.9.7

Дата выхода: 04 июля 2011

Усовершенствования:

Добавлена возможность копирования настроек из одной камеры в одну или несколько других. Делается это с помощью новых кнопок Copy и Paste в самом верху окна.
 
Добавлен раздел Submit Scripts. В этом разделе можно указать файлы скриптов, которые будут запускаться при каждом рендере - скрипт Per Submit запускается один раз перед началом просчета всех камер, Per Camera-скрипт запускается перед просчетом каждой камеры. Эти скрипты выполняются на локальной машине, не на рендер сервере.
 
Добавлено поле Last Rendered Location - в нем хранится путь файла последнего просчета. Удобно вспоминать куда и что рендерилось последний раз.
 
Надеюсь многие пользуются значениями по умолчанию, которые прописываются в INI файл. В этой версии добавлены локальные значения по умолчанию! Теперь можно задавать значения по умолчанию для каждой сцены, проекта или любого другого набора сцен. Для этого надо указать путь к локальному INI файлу в разделе Options, в поле Local INI Path. Путь можно (и нужно) задавать с помощью ключевых слов - например %scenepath% или %projectpath%. Затем нужно нажать кнопку To Local Defaults в нужном разделе, и значения этого раздела будут задаваться для всех новых камер в текущей сцене. Кроме того можно указать путь к локальному INI файлу в глобальном INI файле, причем задавать его надо относительно текущей сцены или проекта использую ключевые слова %scenepath%, %projectpath%, %up%, и тогда в каждой сцена будет локальный INI файл будет подключаться автоматически!

Исправления:

Эксперимент - пустое поле в этой версии считается неустановленным, т.е. пустое поле заменяется значением по умолчанию.
Render Try Counter - теперь начинает считать с нуля.
Render Try Counter больше не сбрасывается при открывании нового файла.
Кнопка Create Folders Silently  перенесена в раздел Options.
Все кнопки Save Defaults переименованны в To Local Defaults. Эти кнопки сохраняют значения раздела в локальный INI файл если указан путь локального файла, в противном случае значения сохраняются в глобальный INI файл.
Исправлены ошибки при сохранении параметров в локальные и глобальные INI файлы.
